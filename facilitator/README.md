## Facilitator Pipeline

```
Advanced cohort completion
        ↓
Facilitator onboarding (Weeks 23–24 of Advanced)
        ↓
Junior facilitator: Basic cohort (supervised)
        ↓
Senior facilitator: Intermediate cohort
        ↓
Curriculum contributor: Write access to mds-curriculum
```

## Access

- **Facilitator guide** (`facilitator/`): Public (this repository)
- **Private facilitator notes**: `facilitator-guide` repo (separate, write access: facilitators only)
