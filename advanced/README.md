# Advanced Tier

**Duration:** 24 weeks  
**Language:** Python + AI-assisted  
**Goal:** Clinical tool builder or preprint-ready researcher

---

## Overview

The Advanced tier produces residents who can build clinical tools or produce preprint-ready research. AI-assisted workflows are introduced critically — residents learn what LLMs cannot catch before they learn to use them.

## Structure

Each week follows the pattern:
```
week-NN/
├── index.qmd          # Resident-facing module
├── facilitator.qmd    # Facilitator notes
└── data/              # Module dataset or pointer (MIMIC-IV pointers only)
```

## Datasets

- MIMIC-IV (PhysioNet credentialed, local copy required — never committed)

## Capstone

Deployed clinical tool prototype **or** preprint-ready manuscript (builder/researcher track).
