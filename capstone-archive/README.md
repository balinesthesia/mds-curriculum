# Capstone Archive

Past cohort capstones published with resident consent.

---

## Structure

```
capstone-archive/
├── basic/
│   └── cohort-NN-
├── intermediate/
│   └── cohort-NN-
└── advanced/
    └── cohort-NN-
```

## Inclusion Criteria

- Resident explicit consent (opt-in, not opt-out)
- Capstone passed assessment rubric
- Code is reproducible (or documented why not)

## Format

Each entry:
- `README.md` — Abstract, methods summary, key findings
- `report.qmd` / `report.pdf` — Final capstone document
- `code/` — Reproducible analysis code
- `data/` — Small datasets or `data/README.md` with access instructions

## Privacy

- No PHI ever committed
- Synthetic or public datasets only
- Cohort identifiers, not individual names
