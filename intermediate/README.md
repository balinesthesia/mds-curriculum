# Intermediate Tier

**Duration:** 20 weeks  
**Language:** Python  
**Goal:** Independent analyst, researcher/builder entry

---

## Overview

The Intermediate tier transitions residents from reproduction to independent analysis. Introduction to prediction vs inference as the central methodological distinction.

## Structure

Each week follows the pattern:
```
week-NN/
├── index.qmd          # Resident-facing module
├── facilitator.qmd    # Facilitator notes
└── data/              # Module dataset or pointer
```

## Datasets

- VitalDB slices
- eICU-CRD samples (PhysioNet credentialed)

## Capstone

Quarto manuscript + reproducible Python pipeline using VitalDB or eICU.
